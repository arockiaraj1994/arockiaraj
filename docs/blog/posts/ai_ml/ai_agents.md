# What is AI Agents?

In simple term, Computer Agent (AI Agent) is something can act like kind of program that can perform some autonomus actions like searching, analysing and taking some decision and provide you the end results. 

Don't think too complex, Lets think the below 

You wanted to book a Cab, There are multiple platform to book a cab (Ola, Uber, Rapido). How to find the low cost and book the cab.

Currently, we need to open all the apps and provide the source and destinaiton and check the price.
Step 1: Open Ola provide the source and destination and check the price. (Remind the price)
Step 2: Open Uber provide the source and destination and check the price. (Remind the price)
Step 3: Open Rapido  provide the source and destination and check the price. (Remind the price)

Two hassale for the above method
1. Time consuming
2. Keep the price in mind
3. There are few repetative operation 


Now, Agent can perform the above operation behalf of you. Just instruct the Agent to do the Cab price analysis and book the van. It can analyse the price and book the cab without human intervention. 

You may think that can be done even in the piece of code, Obsolutly correc but the problem here. Let assume you wrote a piece of code that can analyse and book the cab. After few weeks you wanted to book a cab for later. Now you have to modify your code to introduce the new feature. In the AI Agent can to do for you without making any code changes.


Hope you got some idea about what is Agent.
